## 📊 플랫폼별 데이터 수집 전략

### 1. YouTube (최우선 권장)

#### 🟢 장점
- **높은 데이터 품질**: 긴 형태 컨텐츠로 깊이 있는 분석 가능
- **안정적 API**: Google 공식 API, 높은 신뢰성
- **풍부한 메타데이터**: 조회수, 좋아요, 댓글, 구독자 수 등
- **검색 정확도**: 키워드 매칭 정확성 높음

#### 🔴 단점
- **업로드 빈도**: 상대적으로 낮은 게시 빈도
- **트렌드 반응**: 실시간 트렌드 반영 느림

#### 📈 수집 가능 데이터
```
비디오 메타데이터:
- 제목, 설명, 태그
- 조회수, 좋아요/싫어요, 댓글 수
- 업로드 날짜, 지속시간
- 채널 정보 (구독자 수, 채널명)

댓글 데이터:
- 댓글 텍스트, 작성자
- 댓글 좋아요 수, 답글 수
- 작성 시간
```

#### 🔧 기술적 구현
- **API**: YouTube Data API v3
- **일일 할당량**: 10,000 units
- **수집 방법**: 키워드 검색 → 비디오 상세 → 댓글 수집

---

### 2. Instagram (중간 우선순위)

#### 🟢 장점
- **높은 참여도**: 사용자 상호작용 활발
- **비주얼 컨텐츠**: 제품 이미지 분석 가능
- **인플루언서 생태계**: 브랜드 협업 컨텐츠 많음

#### 🔴 단점
- **API 제한**: Meta 정책으로 인한 접근 제한
- **데이터 제약**: 공개 데이터만 수집 가능
- **개발 복잡성**: API 승인 프로세스 복잡

#### 📈 수집 가능 데이터
```
게시물 메타데이터:
- 캡션, 해시태그
- 좋아요 수, 댓글 수
- 게시 날짜
- 미디어 타입 (사진/비디오)

계정 정보:
- 팔로워 수, 팔로잉 수
- 프로필 정보
- 비즈니스 계정 여부
```

#### 🔧 기술적 구현
- **API**: Instagram Basic Display API / Instagram Graph API
- **제약사항**: 비즈니스 계정 승인 필요
- **수집 방법**: 해시태그 검색 → 게시물 상세

---

### 3. TikTok (낮은 우선순위)

#### 🟢 장점
- **실시간 트렌드**: 최신 트렌드 즉시 반영
- **높은 확산성**: 바이럴 효과 분석 가능
- **젊은 사용자층**: 특정 타겟 그룹 분석

#### 🔴 단점 (주요 제약사항)
- **높은 노이즈**: 중복/광고 컨텐츠 많음
- **API 제한**: 공식 API 접근 어려움
- **데이터 불안정성**: 컨텐츠 수명 짧음
- **품질 일관성**: 동일 컨텐츠 중복 업로드 빈번

#### 📈 수집 가능 데이터
```
비디오 메타데이터:
- 설명, 해시태그
- 조회수, 좋아요, 공유 수
- 업로드 날짜, 지속시간
- 작성자 정보

제한사항:
- 댓글 데이터 수집 어려움
- API 할당량 매우 제한적
```

#### 🔧 기술적 구현
- **API**: RapidAPI TikTok API23 (서드파티)
- **제약사항**: 일일 요청 제한, 높은 비용
- **수집 방법**: 키워드 검색 → 더미 댓글 생성

### 플랫폼별 검색 전략

#### YouTube
```
1차 검색: 제목 + 설명 키워드 매칭
2차 필터: 조회수 1,000 이상
3차 수집: 상위 30개 비디오
4차 분석: 댓글 감정 분석
```

#### Instagram
```
1차 검색: 해시태그 기반
2차 필터: 좋아요 100 이상
3차 수집: 최근 30일 게시물
4차 분석: 캡션 텍스트 분석
```

#### TikTok
```
1차 검색: 일반 검색 + 사용자 검색
2차 필터: 중복 제거
3차 수집: 상위 20개 (API 제한)
4차 보완: 더미 댓글 생성
```

## 🏆 영향력 지표 설계

### 인플루언서 영향력 점수 공식
```
영향력 점수 = (팔로워 수 × 0.3) + (평균 조회수 × 0.4) + (참여율 × 0.3)

참여율 = (좋아요 + 댓글 + 공유) / 조회수 × 100
```

### 채널/계정 등급 분류
- **Tier 1 (메가 인플루언서)**: 100만+ 팔로워
- **Tier 2 (매크로 인플루언서)**: 10만-100만 팔로워  
- **Tier 3 (마이크로 인플루언서)**: 1만-10만 팔로워
- **Tier 4 (나노 인플루언서)**: 1천-1만 팔로워

### 브랜드 영향력 지표
- **언급 빈도**: 키워드별 월간 언급 횟수
- **감정 점수**: 긍정/중립/부정 비율
- **확산 지수**: 좋아요/공유 대비 조회수 비율
